# phone_book
Простецкий телефонный справочник для тренировки в Питоне

Телефонный справочник, который сохраняет контакты в файл и имеет следующий функционал:

- открыть файл
- сохранить файл
- показать все контакты
- создать контакт
- найти контакт
- изменить контакт
- удалить контакт
- выход

Справочник запускается из файла ph_b_controller.py
Все три файла (ph_b_controller.py, ph_b_model.py, ph_b_view.py) должны находиться в одной директории для работы программы.
ph_b_controller.py - файл упраления поведение программы
ph_b_model.py - реализованные в программе алгоритмы обработки данных
ph_b_view.py - интерфейс взаимодействия с пользователем.
